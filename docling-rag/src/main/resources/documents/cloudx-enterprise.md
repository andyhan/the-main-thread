# CloudX Enterprise Platform — Sales reference

## Enterprise tier overview

The **Enterprise** tier of CloudX is designed for large organizations that need advanced security, dedicated support, and scalable multi-region deployment.

## Enterprise tier features

- **Single sign-on (SSO)** with SAML 2.0 and OpenID Connect, including SCIM user provisioning.
- **Dedicated virtual private cloud (VPC) connectors** for hybrid and on-premises data sources.
- **99.99% uptime SLA** with priority incident response and a named Technical Account Manager (TAM).
- **Advanced audit logging** with immutable retention and export to your SIEM (Splunk, Elastic, Microsoft Sentinel).
- **Data residency controls** so primary data and backups stay in customer-selected regions.
- **Unlimited API throughput** within fair-use policy; burst capacity negotiated per contract.
- **24×7 premium support** with a 1-hour response time for severity-1 issues.

## Positioning vs other tiers

Enterprise includes everything in **Professional**, plus VPC connectors, SSO/SCIM, the uptime SLA, TAM, and compliance-oriented logging. **Starter** is self-serve only and does not include SSO or VPC options.

## Common buyer questions

- **“Do we get a sandbox?”** Yes — Enterprise includes a non-production environment matching production topology for testing integrations.
- **“Can we use our own KMS keys?”** Yes — customer-managed keys (CMK) are supported in Enterprise.
